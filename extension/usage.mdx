The ApplyTrack extension works silently in the background, capturing every application you submit. No manual entry, no extra clicks—just apply and watch your dashboard grow automatically.

## How It Works

ApplyTrack uses a high-speed content script that intelligently monitors your job application activity:

<Steps>
  <Step title="Plug in the Bridge">
    Once installed, the browser extension connects your application flow to your ApplyTrack dashboard.
    
    The extension icon in your toolbar indicates active status with a green checkmark.
  </Step>
  
  <Step title="Automated Intelligence">
    Every time you click "Submit" on a job application, the extension instantly captures metadata:
    
    - Company name and job title
    - Application date and time
    - Job description and requirements
    - Salary range (if available)
    - Location and work type (remote/hybrid/onsite)
    - Application URL and job board source
    
    This happens automatically—no manual data entry required.
  </Step>
  
  <Step title="Master Your Results">
    The captured data syncs immediately to your dashboard where you can:
    
    - View real-time response rates across all applications
    - Track recruiter activity and email updates
    - Receive AI-suggested next steps based on your pipeline
    - Monitor interview schedules and follow-ups
  </Step>
</Steps>

<Note>
The content script only activates on supported job boards. It remains dormant on other websites to preserve browser performance.
</Note>

## Applying for Jobs

### Standard Application Flow

The extension works seamlessly with your existing job search workflow:

1. **Browse** any supported job board (LinkedIn, Indeed, Glassdoor, etc.)
2. **Complete** the application form as you normally would
3. **Submit** your application by clicking the final submit button
4. **Confirm** the green success notification from ApplyTrack (appears briefly in the corner)
5. **View** the application immediately in your [dashboard](https://app.applytrack.ai/dashboard)

<Tip>
You don't need to keep the ApplyTrack dashboard open. The extension captures applications even when the dashboard is closed.
</Tip>

### What Gets Captured

The extension intelligently extracts key information from your applications:

| Data Field | Description |
|------------|-------------|
| **Company Name** | Organization name, verified against company database |
| **Job Title** | Position title and level (Junior, Senior, Lead, etc.) |
| **Application Date** | Timestamp of submission |
| **Job Description** | Full description text for AI analysis |
| **Requirements** | Skills, experience, and qualifications listed |
| **Salary Range** | Compensation info if disclosed by employer |
| **Location** | City, state/country, and remote status |
| **Job URL** | Direct link back to the original posting |
| **Platform** | Source job board (LinkedIn, Indeed, etc.) |
| **Application Method** | Quick Apply, Easy Apply, or Full Application |

The extension works seamlessly with ApplyTrack's email sync feature:

- **Automatic Updates**: When recruiters respond via email, your application status updates automatically
- **Thread Tracking**: All email conversations are linked to the specific application
- **Interview Scheduling**: Calendar invites are detected and added to your timeline

<Tip>
Connect Gmail or Outlook in your [dashboard settings](https://app.applytrack.ai/settings) for automatic email tracking.
</Tip>

## Privacy and Data Security

ApplyTrack prioritizes your privacy:

- **Encrypted Transmission**: All data is encrypted with TLS 1.3 during sync
- **Local Storage**: Applications are cached locally before upload to prevent data loss
- **Selective Access**: The content script only activates on job board domains
- **No Tracking**: We never track your browsing activity outside of application submissions

<Warning>
The extension requires access to job board websites to function. We do not access or store passwords, personal messages, or unrelated browsing data.
</Warning>

## Extension Settings

Click the extension icon and select **Settings** to customize:

- **Notification preferences**: Control when and how you're notified
- **Auto-sync frequency**: Choose between instant, hourly, or manual sync
- **Platform filters**: Disable tracking on specific job boards if needed
- **Data fields**: Select which fields to capture (all enabled by default)
- **Privacy mode**: Opt to blur sensitive data in notifications

## Troubleshooting

### Application Not Captured

1. **Verify platform support**: Check the [supported platforms list](/extension/supported-platforms)
2. **Check extension status**: Ensure icon shows green (active) state
3. **Wait 1-2 minutes**: Some applications may take time to process
4. **Refresh dashboard**: Click the sync button in your dashboard
5. **Manual entry**: As a fallback, you can always add applications manually

### Duplicate Applications

1. **One-time sync issue**: Delete the duplicate from your dashboard
2. **Multiple submissions**: Verify you didn't submit to the same job twice
3. **Clear cache**: Open extension settings and clear cached data

### Incorrect Information

1. **Edit in dashboard**: All captured data can be manually edited
2. **Report issue**: Help us improve by reporting incorrect captures
3. **Platform bug**: Some job boards may display inconsistent data

### Extension Performance

1. **High CPU usage**: Disable on platforms you don't use via settings
2. **Slow page loads**: The extension has minimal impact, but try disabling temporarily
3. **Browser crashes**: Update to the latest browser version
