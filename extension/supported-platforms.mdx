The ApplyTrack extension uses intelligent content scripts to detect successful applications across 200+ job boards and career portals. Our platform support is constantly expanding based on user requests and job market trends.

## Coverage Overview

ApplyTrack automatically tracks applications on:

- **Major Job Boards**: LinkedIn, Indeed, Glassdoor, and more
- **Tech-Specific Platforms**: AngelList, Dice, Stack Overflow Jobs
- **Industry Portals**: Specialized boards for healthcare, education, finance, etc.
- **Company ATS Systems**: Greenhouse, Lever, Workday, and 50+ others
- **Regional Boards**: Country and region-specific platforms worldwide
- **Niche Sites**: Remote-only, freelance, and gig economy platforms

<Note>
If a platform you use isn't supported yet, you can request it at [feedback@applytrack.ai](mailto:feedback@applytrack.ai). We prioritize new integrations based on user demand.
</Note>

### Detection Reliability

| Tier | Description | Example Platforms |
|------|-------------|-------------------|

## Platform-Specific Notes

### LinkedIn

- Supports Easy Apply and external redirects
- Tracks InMail conversations with recruiters
- Captures company insights and employee connections
- Premium features (e.g., "Featured Applicant") are noted in your dashboard

### Indeed

- Indeed Apply captured instantly
- Company redirects tracked when you return to Indeed
- Employer responses and interview requests auto-update application status
- Indeed Resume applications linked to your profile

### Glassdoor

- Easy Apply detection
- Company ratings automatically attached to applications
- Salary estimates linked in dashboard
- Interview reviews accessible from application view

### Company ATS Systems

- Best results when you complete entire application in one session
- Some systems require reaching the confirmation page
- Multi-page applications tracked when final submit is clicked
- File uploads (resume, cover letter) are noted but not stored by ApplyTrack

## Troubleshooting Platform Issues

### Platform Not Listed

1. **Search this page**: Use Ctrl+F (Cmd+F on Mac) to search for the platform name
2. **Check ATS**: The company may use a supported ATS (Greenhouse, Lever, etc.)
3. **Request integration**: Email us to add support
4. **Manual entry**: Add the application manually as a workaround

### Application Not Detected

1. **Verify platform support**: Ensure it's on this list
2. **Check extension status**: Green icon = active and ready
3. **Complete full submission**: Some platforms require the confirmation page
4. **Wait 1-2 minutes**: Sync may take a moment on slower connections
5. **Report issue**: Help us improve detection by reporting the problem

### Incorrect Data Captured

1. **Edit in dashboard**: All fields can be manually corrected
2. **Report issue**: Send us the job URL so we can fix the detection
3. **Check job posting**: Sometimes job boards display inconsistent data

<Warning>
If a supported platform consistently fails to detect your applications, please contact support with the job board name and example job URL.
</Warning>
