## Design Features

### Interactive Elements

- **Hover Effects**: Stat cards feature smooth transitions and scale animations
- **Color Coding**: Each metric uses distinct colors (blue, amber, emerald, indigo)
- **Border Animations**: Cards highlight on hover with blue-tinted shadows
- **Live Status Indicators**: Red pulsing dots for urgent interviews

### Visual Styling

- **Rounded Corners**: Large border radius (2rem, 2.5rem) for modern aesthetic
- **Soft Shadows**: Subtle elevation with blue-tinted shadows
- **Typography**: Black font weights for primary text, uppercase tracking for labels
- **Icon System**: Lucide React icons with contextual coloring
